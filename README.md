## Spring Boot 项目结构

+ config 配置类
+ controller 控制层
+ entity 实体类: 与数据库表一一对应
+ model: 对entity中的数据进行解释和封装
+ domain: 对model中的数据进行整合, 最终发送给前端
+ interceptor 拦截器: 
+ mapper: 用以 mybatis 映射数据库 
+ service: 处理业务逻辑
